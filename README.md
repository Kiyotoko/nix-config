# ❄️ My NixOS Config

```bash
sudo nixos-rebuild switch --flake .
home-manager switch --flake .
```

## Software

- Display manager: Hyprland
- Screenshot utility: Hyprshot
- Locker: Swaylock
- Wayland Bar: Waybar
- Terminal emulator: Alacritty
- File manager: Nemo
- Browser: Librewolf
- App menu: Wofi
- Editor: VS Codium
